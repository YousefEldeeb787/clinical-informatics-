## ? What's Working

### Authentication Flow ?
1. User enters credentials on login page
2. `authService.login()` sends POST to `/api/auth/login`
3. Backend validates credentials with BCrypt
4. Backend generates JWT token
5. Token stored in localStorage
6. All subsequent requests include token in Authorization header
7. Backend validates token on protected routes

### Data Flow Example (Appointments) ?
1. User clicks "Appointments" menu
2. Component mounts, `useEffect` triggers
3. Calls `appointmentsService.getAllAppointments()`
4. API client adds JWT token to request
5. Sends GET to `http://localhost:5160/api/appointments`
6. Backend validates token
7. AppointmentsController calls AppointmentService
8. AppointmentService queries database via Repository
9. Data returned as JSON
10. Frontend receives response
11. Redux store updated with `setAppointments()`
12. Component re-renders with data

### Authorization (RBAC) ?
- Admin: Full access to all features
- Clinician: Patient care, medical records, surgeries
- Receptionist: Appointments, patient info, billing
- Patient: Own information, appointments, prescriptions
